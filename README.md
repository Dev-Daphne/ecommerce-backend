# 🛒 E-commerce Backend API

REST API built with Node.js and Express for product management.

---

## 🚀 Technologies

- Node.js
- Express
- JavaScript

---

## 📌 Features

- 📦 Get all products
- ➕ Create a new product
- ✏️ Update a product
- 🗑 Delete a product

---

## 📡 API Endpoints

| Method | Endpoint            | Description          |
|--------|--------------------|----------------------|
| GET    | /produtos          | Get all products     |
| POST   | /produtos          | Create a product     |
| PUT    | /produtos/:id      | Update a product     |
| DELETE | /produtos/:id      | Delete a product     |
